wasm32-unknown-unknown

wasm32 指 cpu类型与寻址空间大小，32 指 32bit 的内存寻址空间，64 指 64bit 的内存寻址空间（目前没有实现）。

第一个 unknown，指对程序进行编译的平台（任意）；第二个 unknown，指运行目标程序的平台（任意）。
